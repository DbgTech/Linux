
0. 添加开启GDB stub的设置（bochsrc.bxrc）

gdbstub: enabled=1,port=1234

1. 启动 bochs

$ bochs -q -f bochsrc.bxrc

-q 安静模式，不提示菜单
-f 制定要读取的配置文件，默认读取 ".bochsrc"


2. gdb启动

首先链接到bochs的gdbstub上：

gdb> target remote locahost:1234  
或
gdb> target remote :1234


在指定地址上设置断点，比如引导扇区加载位置  0x7c00

gdb> gdb *0x7c00

设置汇编格式（架构格式）为16位

gdb> set archtecture i8086
gdb> set archtecture i386

显示下一条汇编语句：

gdb> disp /i $cs * 16 + $pc	// 16位汇编
gdb> disp /i $eip		// 32位保护模式汇编
